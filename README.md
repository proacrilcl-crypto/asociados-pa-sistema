# Asociados P&A - Sistema de Gestión

Versión conectada a Firebase.

## Incluye
- Login con Firebase Auth
- Pedidos con numeración automática 01, 02, 03...
- Estados completos
- Abonos y delivery
- Ingreso disponible solo en Entregado o Retirado
- Bloqueo de edición/eliminación en pedidos finalizados
- Gastos
- Dashboard
- Comisiones al 10%
- Orden de trabajo PDF

## Cómo usar
1. Sube estos archivos al repositorio conectado con Vercel.
2. Vercel desplegará automáticamente.
3. Inicia sesión con los usuarios creados en Firebase.
